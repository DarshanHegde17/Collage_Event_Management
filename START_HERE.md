## 🔧 Troubleshooting

### MongoDB Not Running?
```bash
net start MongoDB  # Windows
sudo systemctl start mongodb  # Linux
```

### Port 8080 Busy?
Edit `application.properties`:
```properties
server.port=8081
```

### Build Fails?
```bash
mvn clean install -U
```
